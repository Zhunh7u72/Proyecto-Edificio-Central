# Proyecto Edificio Central — UTN

Portal web del Edificio Central de la Universidad Técnica del Norte (FEUE): noticias, eventos, galería, documentos y panel administrativo.

## Getting Started

```bash
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en el navegador.

## Stack

- Next.js 16 (App Router)
- Supabase
- TypeScript
